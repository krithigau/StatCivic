#  StatCivic: Intelligent Grievance Redressal System

> **Bridging the gap between Citizens and Administration through AI-driven governance.**

##  Overview
**StatCivic** is a modern, full-stack web application designed to streamline the chaotic process of civic grievance reporting. It replaces outdated, confusing government portals with a **smart, transparent, and responsive** platform.

The core innovation is the **Smart Routing Engine**, which analyzes user descriptions in real-time to automatically categorize complaints (e.g., "Streetlight broken" -> "Electricity Board") without human error.

---

##  Key Features

###  For Citizens
* **One-Click Login:** Secure OTP-based authentication (No passwords required).
* **Smart Auto-Fill:** The "AI" detects the department based on keywords as you type.
* **Real-Time Tracking:** View live status updates (Pending, In Progress, Resolved).
* **Accountability Tools:** Ability to **Reopen** closed tickets with a reason if the fix was unsatisfactory.

###  For Administrators
* **Command Center:** A unified dashboard to view all citizen issues.
* **Data Visualization:** Live Pie Charts (Dept Breakdown) and Bar Graphs (Priority Analysis).
* **Priority Flags:** Auto-detection of "Critical" keywords (Fire, Accident, Dengue).
* **Reopen Alerts:** Special pulsing badges for reopened tickets to highlight recurring issues.
